\frametitle{Кинематика}
Пресмятане на скоростта:

\begin{equation}
 \mathrm{ v_2 = \frac{x_3 - x_1}{ t_3 - t_1} }, \quad \mathrm{ v_3 = \frac{x_4 - x_2}{ t_4 - t_2} },\quad  \dots, \mathrm{ v_n = \frac{x_{n+1} - x_{n-1}}{ t_{n+1} - t_{n-1}} }
\end{equation}

Пресмятане на ускорението:

\begin{equation}
 \mathrm{ a_2 = \frac{v_3 - v_1}{ t_3 - t_1} }, \quad \mathrm{ a_3 = \frac{v_4 - v_2}{ t_4 - t_2} },\quad  \dots, \mathrm{ a_n = \frac{v_{n+1} - v_{n-1}}{ t_{n+1} - t_{n-1}} }
\end{equation}
